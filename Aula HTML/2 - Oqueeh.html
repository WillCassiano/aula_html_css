O que é HTML?

HTML é a linguagem de marcação padrão para a criação de páginas da Web.

  • HTML significa Hyper Text Markup Language
  • HTML descreve a estrutura de uma página Web
  • HTML é composto por uma série de elementos
  • elementos HTML dizer ao navegador como exibir o conteúdo
  • elementos HTML são representados por tags
  • tags HTML rotular peças de conteúdo, como "título", "parágrafo", "mesa", e assim por diante
  • Browsers não exibir as tags HTML, mas usá-los para processar o conteúdo da página
